SEM instalar nada! Vamos usar Supabase (PostgreSQL em nuvem).

PASSO 1: Abrir Browser
═════════════════════════════════════════════════════════════════
https://supabase.com

PASSO 2: Sign Up
═════════════════════════════════════════════════════════════════
Clicar: "Sign Up"
Usar: GitHub ou Google (mais rápido)

PASSO 3: Criar Projeto
═════════════════════════════════════════════════════════════════
1. "New project"
2. Name: shopvivaliz-medusa
3. Password: medusa123456
4. Region: São Paulo (sp-1)
5. Clicar: "Create new project"
6. AGUARDAR 2-3 MINUTOS (diz "Building your database...")

PASSO 4: Copiar Connection String
═════════════════════════════════════════════════════════════════
1. Settings (engrenagem) → Database
2. Connection Strings
3. Copy o URI (a string que começa com postgresql://)

Vai parecer assim:
postgresql://postgres:XXXXXXX@db.YYYYYYY.supabase.co:5432/postgres

PASSO 5: Colar no Arquivo
═════════════════════════════════════════════════════════════════
Abrir: C:\Users\user\site-shopvivaliz\claude\medusa\apps\backend\.env

Procurar por:
DATABASE_URL=postgresql://medusa:medusa123@localhost:5432/shopvivaliz_medusa

Substituir INTEIRA por:
DATABASE_URL=postgresql://postgres:medusa123456@db.YYYYYYY.supabase.co:5432/postgres

(Cole exatamente como copiou do Supabase)

═════════════════════════════════════════════════════════════════
QUANDO TERMINAR:
═════════════════════════════════════════════════════════════════

Execute este comando (depois de colar DATABASE_URL):

cd C:\Users\user\site-shopvivaliz\claude\medusa\apps\backend
