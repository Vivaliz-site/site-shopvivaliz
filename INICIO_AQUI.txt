PRÓXIMOS PASSOS (5 MINUTOS):

1️⃣ OBTER BANCO DE DADOS

   OPÇÃO A - Supabase (RECOMENDADO - sem instalar nada):
   • Abrir: https://supabase.com
   • Sign up → New Project → São Paulo → Aguardar 2 min
   • Settings → Database → Connection Strings → Copiar URL

   OPÇÃO B - PostgreSQL Local:
   • Instalar: https://www.postgresql.org/download/windows/
   • psql -U postgres
   • CREATE DATABASE shopvivaliz_medusa;
   • CREATE USER medusa PASSWORD 'medusa123';
   • GRANT ALL PRIVILEGES ON DATABASE shopvivaliz_medusa TO medusa;

   OPÇÃO C - Banco Existente (HostGator):
   • Usar connection string do banco remoto

2️⃣ RODAR SCRIPT DE SETUP

   PowerShell (como admin):
   .\RODAR_AGORA.ps1

   Cole a DATABASE_URL quando pedir.
   Aguarde completar (migrations + seed data).

3️⃣ INICIAR BACKEND

   Abrir Terminal 1:
   cd claude\medusa\apps\backend
   npm run dev

   Aguarde: "Server listening on port 9000"
   Acessar: http://localhost:9000/admin

4️⃣ INICIAR STOREFRONT

   Abrir Terminal 2:
   cd claude\medusa\apps\storefront
   npm run dev

   Aguarde: "Ready in Xs"
   Acessar: http://localhost:3000

5️⃣ TESTAR FLUXO

   Admin (localhost:9000/admin):
   • Login: admin@medusajs.com / supersecret
   • Criar produto

   Storefront (localhost:3000):
   • Ver produto
   • Adicionar ao carrinho
   • Checkout
