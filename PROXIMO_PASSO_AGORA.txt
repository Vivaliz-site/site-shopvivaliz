ESCOLHA UMA OPÇÃO:

OPÇÃO A - DOCKER (MAIS RÁPIDO) ⭐
═══════════════════════════════════════════════════════════════

Se tiver Docker instalado:

1. Rodar Docker Compose:
   docker-compose up -d

2. Aguardar banco ativar (30 segundos)

3. Configurar Backend:
   cd claude\medusa\apps\backend

   (edite .env e substitua)
   DATABASE_URL=postgresql://medusa:medusa123@localhost:5432/shopvivaliz_medusa

4. Rodar Migrações:
   npm run migrate
   npm run seed

5. Terminal 1 - Backend:
   npm run dev
   (esperado: "Server listening on port 9000")

6. Terminal 2 - Storefront:
   cd ..\..\apps\storefront
   npm run dev
   (esperado: "Ready in Xs")

7. Acessar:
   Admin: http://localhost:9000/admin
   Storefront: http://localhost:3000

OPÇÃO C - PostgreSQL LOCAL
═══════════════════════════════════════════════════════════════

Se já tiver PostgreSQL instalado:

1. Criar banco:
   psql -U postgres
   CREATE DATABASE shopvivaliz_medusa;
   CREATE USER medusa PASSWORD 'medusa123';
   GRANT ALL PRIVILEGES ON DATABASE shopvivaliz_medusa TO medusa;
   \q

2. Configurar: claude\medusa\apps\backend\.env
   DATABASE_URL=postgresql://medusa:medusa123@localhost:5432/shopvivaliz_medusa

3. Rodar migrations e servers (como acima)
