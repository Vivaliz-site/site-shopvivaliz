PASSO 1: Abra PowerShell COMO ADMINISTRADOR
───────────────────────────────────────────────────────────────────────────────
1. Clique em [Windows]
2. Digite: PowerShell
3. Clique direito → "Executar como administrador"


PASSO 2: Execute o script de setup
───────────────────────────────────────────────────────────────────────────────
cd C:\site-shopvivaliz
.\SETUP-COMPLETE.ps1

Aguarde ~15 minutos. O script vai:
  ✓ Instalar Docker Desktop
  ✓ Instalar Ollama
  ✓ Baixar modelo Mistral 7B (4.1 GB)
  ✓ Testar tudo automaticamente


PASSO 3: Em novo terminal, rode o servidor Ollama
───────────────────────────────────────────────────────────────────────────────
Abra NOVO PowerShell (não precisa admin):

Mantenha este terminal ABERTO enquanto trabalhar.


PRONTO! 🎉 Seu sistema está operacional!
