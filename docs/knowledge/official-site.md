# Site Oficial como Fonte de Referência

## Fonte

O domínio oficial `https://www.shopvivaliz.com.br` é uma fonte válida para conteúdo institucional e comercial do ShopVivaliz.

Data de verificação desta referência: 11 de julho de 2026.

## Conteúdo confirmado

O site oficial pode ser usado como base para:

- termos e condições;
- informações institucionais;
- central de ajuda;
- atendimento;
- meios de pagamento divulgados;
- categorias comerciais apresentadas no storefront;
- textos de navegação e rodapé.

## Rotas oficiais mapeadas

- `/p/termos`
- `/p/quem-somos`
- `/p/ajuda`
- `/p/atendimento`

## Meios de pagamento divulgados

- Visa
- Mastercard
- Boleto
- Depósito
- Pix

## Categorias destacadas

- Casa, Móveis e Decoração
- Ferramentas
- Construção
- Acessórios para Veículos
- Pet Shop
- Arte, Papelaria e Armarinho

## Regras para agentes e desenvolvedores

1. Usar o site oficial como evidência para conteúdo institucional e comercial.
2. Registrar a data da consulta quando o conteúdo for incorporado ao repositório.
3. Não substituir dados técnicos, banco, logs ou contratos de API por conteúdo visual do storefront.
4. Em caso de divergência entre o site oficial e o código, registrar a divergência antes de alterar produção.
5. Não inferir informações que não estejam publicadas ou confirmadas.
