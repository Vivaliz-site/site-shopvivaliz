PASSO 1: Abra Terminal/PowerShell AQUI na PC e execute:

node scripts/ios-command-listener.js

Você vai ver:
  ✅ 🎯 iOS Command Listener rodando na porta 9999
  ✅ 📱 Acesse do iPhone: http://seu-pc-ip:9999/execute

═══════════════════════════════════════════════════════════════════

PASSO 2: Do iPhone, abra Safari e copie este comando:

curl -X POST http://SEU-PC-IP:9999/execute \
  -H "X-Token: hBu-3gs3meFOp82AnXLzljmIvNaf-7ih" \
  -H "Content-Type: application/json" \
  -d '{"cmd":"git status"}'

(Substitua SEU-PC-IP pelo IP real, ex: 192.168.1.100)

ERROS COMUNS:

❌ "Connection refused"
   → Listener não está rodando (execute node command primeiro)

❌ "Invalid token"
   → Token errado (use: hBu-3gs3meFOp82AnXLzljmIvNaf-7ih)

❌ "ERR_NAME_NOT_RESOLVED" (iPhone)
   → IP errado (execute ipconfig pra descobrir)

❌ "Connection timed out"
   → Firewall bloqueando porta 9999
      → Windows Defender > Firewall > Allow app > Node.js

═══════════════════════════════════════════════════════════════════

FIREWALL - Se não conectar do iPhone:

Windows Defender Firewall:
1. Win + R → wf.msc
2. Inbound Rules → New Rule
3. Port → TCP → 9999
4. Allow connection
5. Nome: "iOS Command Listener"

Ou via PowerShell:
New-NetFirewallRule -DisplayName "iOS Commands" `
  -Direction Inbound -Action Allow -Protocol TCP -LocalPort 9999
